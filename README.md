# StudyVault

StudyVault is a Notes and Resource Marketplace designed for students to upload, search, download, and share academic resources.

## Features
- User Authentication
- Notes Upload
- Resource Search
- Downloads
- Ratings & Reviews

## Tech Stack
- HTML
- CSS
- JavaScript
- PHP/Flask
- MySQL
